# tanstack-start-devjoke
Final project created for COMP3013, winter 2026(jan-mar) BCIT. This application is using Tanstack Start(Framework) + neon(DB) + Drizzle(ORM) + BetterAuth. Here developer can signup, signin and add, delete, voteup, votedown the jokes. The most votedup joke will display in top three jokes section.
